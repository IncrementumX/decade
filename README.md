# Decade — Investment Case: Tax-Exempt Assets in Brazil

Investment conviction on the role of Brazil's tax-exempt assets — FIIs, Infra Debêntures, FI-Infra, FIAGRO, CRIs, CRAs — in client portfolios.

**Live:** https://incrementumx.github.io/decade/

## Structure

```
index.html              the case (CSS-only tabs — works without JS)
assets/css/styles.css   styles
assets/js/app.js        interactivity (Chart.js charts, tab logic, helpers)
assets/cv.pdf           CV
data/anbima.json        data placeholder
```

Chart.js is loaded from CDN (cdnjs); an internet connection is needed for the charts to render.

## Local preview

```
python -m http.server 8000 --directory .
# then open http://localhost:8000/
```
