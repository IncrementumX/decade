# src/ — fonte editável do index.html

`index.html` (servido pelo Pages) é um **artefato montado**. NÃO edite ele à mão.

- Edite `shell.html` (head, masthead, nav de abas, scripts, footer) ou `sections/NN-*.html` (conteúdo de cada aba).
- Rode `python src/build.py` → regenera `../index.html`.
- Valide o render local e siga o fluxo local-first (CLAUDE.md §5) antes de publicar.

Fragmentos (tamanho aprox.): intro ~4KB · exec ~15KB · landscape ~172KB · assets ~269KB · portfolio ~29KB · references ~6KB · appendix ~3KB.
