CaveatRolling windows are overlapping and descriptive, not independent probabilities. Read each figure as the in-sample frequency with which one asset beat another over that horizon — not an out-of-sample odds estimate. Pairwise hit-rates versus CDI are identical in nominal and real terms (both series deflated by the same inflation factor), so the BRL/USD view changes only the FX-converted comparison, not the vs-CDI ranking.
