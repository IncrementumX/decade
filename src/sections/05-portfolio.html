Guardrails · ranges + caps, not final weights
Concentration caps
Per-issuer cap (single CRI / CRA / debênture) and per-class cap. FII ≤ ~40% of the exempt sleeve — beyond that, drawdown and risk-adjusted return both deteriorate (the Aggressive read). FIAGRO held to a satellite ≤8% of the sleeve.
Illiquidity budget
Cap gated / illiquid sleeves (open-end FI-Infra with redemption queues, pulverized CRA, FIDC-taxed structures) so the occasional-access need is always met from the CDI buffer + listed / secondary-liquid names. Conservative keeps a ~24%-of-sleeve cash buffer; do not let it fall below the access need.
Duration budget
The carry core (IMA-B5+ proxy) is long-duration — it drives both the return and the drawdown. Size the long-duration sleeve to a duration ceiling so a real-rate shock (the −20.9% IMA-B5+ drawdown) stays inside tolerance. A CDI+ credit satellite is a duration-light offset.
Exit-value caveat
NAV-smoothed / illiquid credit understates true mark-to-market vol. The proxy indices (IFIX, IMA-B family) are listed / liquid and show more honest vol than a closed FI-Infra or direct-CRI book would print. Real exit value under stress (gate, bid-ask, queue) can be worse than the proxy DD suggests — do NOT read the proxy vol / DD as the floor for an illiquid implementation.
Rules-engine precedence
Sizing ranges feed the rules engine; its precedence (eligibility → portfolio-fit → concentration / liquidity caps) governs — NOT a point optimizer. Mean-variance over NAV-smoothed credit self-invalidates (vol illusion).
Bottom line (decision-grade)
Within the onshore exempt sleeve, on observed proxies, 2004–2026 was a regime where CDI was a punishingly high hurdle — a diversified exempt-FI book beat cash only modestly in real terms while taking real drawdown. The case is risk-adjusted, not return-maximizing: a Conservative mix delivers ~IMA-B5+ real return at materially lower vol and drawdown. FII adds diversification but is the drawdown driver. The decisive caveats are illiquidity / exit-value and the per-name data gap: the proxy backtest cannot price manager alpha or the exempt tax edge, and an illiquid implementation can exit worse than the listed proxies print. No final sizing is set here; ranges and guardrails only, pending per-name total-return data and CRI/CRA spread validation. Percentage-of-wealth sizing and the model portfolio are completed in the next round.
